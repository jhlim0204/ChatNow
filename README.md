# ChatNow

Realtime chat application using React, Reactstrap, Node.js, Express, Socket.io and MongoDB.

## Installation

You should have Node.js and NPM/Yarn installed on your computer first.

```
yarn
yarn build
node server

Go to localhost:3001
```

If you are using NPM,
```
npm install
npm run build
node server

Go to localhost:3001
```
